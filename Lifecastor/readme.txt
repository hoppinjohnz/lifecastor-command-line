Lifecastor v0.3
Nov. 11, 2012

Instructions:

1. Unzip the Lifecastor.zip to a folder. The folder can be your Downloads folder.

2. Open the command line prompt and change to that directory. There will be 4 files: 
    plann.properties, 
    deterministic-plan.properties, 
    lifecastor, and this 
    readme.txt. 

3. Rename 'lifecastor' to 'lifecastor.exe'.

4. Type 'lifecastor.exe' to run the program. Type 'lifecastor.exe -h' for help.

5. Use options like -b, -t, -v and/or -c. You can combine them too.

6. Use any text editor to modify your properties in the .properties files. By default, it will use plan.properties file. Type 'lifecastor.exe deterministic-plan.properties' to use the other properties file. Feel free to use your own properties files.


You can run endless what-ifs about your own financial life forecast.
